## 📗 プロジェクトの概要

jQueryで作成した自動で画像をスライドさせるプラグインです。

## 🌐 Demo

#### ■ フロント画面（React）

https://isystk.github.io/jquery-autoSlider

## 💬 使い方

各種デーモンを起動する
```
$ npm install
$ npm run dev
```
